## Requirements

- PHP 8.4+
- Composer

## Create a New App

```bash
composer create-project --prefer-dist minicli/app myapp
cd myapp
```

## Run the App

```bash
./minicli help
```

This prints the command list and confirms your installation.

## Project Structure (Typical)

```text
app/
  Commands/
  Middlewares/
  Services/
config/
  *.php
minicli
```
